#Projeto ATIVIDADE de API - DJANGO - SENAI
O projeto se baseia numa API DJANGO, que usa REST FRAMEWORK e SQLITE. Também utiliza POSTMAN para visualização dos ENDPOINTS
A API está documentada e as COLLECTIONS também.

# Qual é a funcionalidade da API?
A API faz interações básicas entre os seguintes models:

* Aluno - (POST, GET, PUT, GET LIST, DELETE)
* Disciplina (POST, GET, PUT, GET LIST, DELETE)
* Tarefa (POST, GET, PUT, GET LIST, DELETE)

# Como Utilizar

* Primeiro, baixe o arquivo usando Git clone https://github.com/GtOliv3r/Atividade_API.git
* Abra o arquivo no VS Code
* Aperte o ctrl+j, para abrir o terminal.
* Digite " pip install -r requirements.txt " e aperte ENTER
* Digite " python manage.py runserver " e aperte ENTER
* Agora faça bom proveito utilizando o POSTMAN. Entre os arquivos baixados, há uma collection de POSTMAN, lá 
consta todos os endpoints, com exemplos de respostas. Clique em cada um e visulize com atenção.
* Obrigado e até mais!
